# Yourscent Import

Catálogo responsivo de perfumes com conversão automática de dólar para real.

## Funcionalidades
- Layout responsivo (mobile, tablet e desktop)
- Listagem em grade adaptável
- Filtro por categoria
- Conversão USD -> BRL em tempo real

## Como usar
1. Baixe o projeto
2. Abra index.html
3. Personalize os produtos no script.js
